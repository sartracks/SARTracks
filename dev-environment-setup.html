SARTracks Dev Environment Setup

From a clean Windows 7 machine (2/6/2012):

Setup git

  1. GitHub setup.
  2. git clone git@github.com:sartracks/SARTracks \sartracks

Setup build environment

  1. Install and run Microsoft Web Platform Installer
  2. Use platform installer to install ASP.NET MVC 3, Sql Expres 2008 R2, IIS 7.5 Express, .NET 4.0, SQL Server Management Studio (optional)
  3. Download and install EF June CTP 2011 (only need EF part)
  4. Add %WINDOWS%\Microsoft.NET\Framework\v4.0.30319 to PATH environment variable.
  5. If Visual Studio not installed: copy WebApplication targets from sartracks\build\msbuild\microsoft to \Program Files\MSBuild ("Program Files (x86)" as needed).

Build

  1. cd \sartracks\code\website
  2. msbuild

Run

  1. "\Program Files\IIS Express\iisexpress.exe" /path:c:\sartracks\code\website /port:25489
  2. Point browser to http://localhost:25489
